# プロジェクト名: Disney-Inspired Pachinko "Magic Rush" (Prototype)

## 1. 目的
React + PixiJS を使用し、パチンコの「液晶演出」の核となる部分を試作する。
「抽選 → 演出開始 → カットイン → 当落告知」の一連の流れをヌルヌル動かす。

## 2. 演出要件（UI/UX最優先）
- **通常時**: 画面中央に数字（図柄）が変動している。
- **抽選**: ボタン（Spaceキー）押下で 1/319 の抽選を行う。
- **演出**: 
  - 内部的に「当たり」の場合：80%の確率で「金カットイン（激アツ）」を発生させる。
  - 内部的に「外れ」の場合：5%の確率で「金カットイン」を出し、期待させてから外す（釣り演出）。
- **ビジュアル**:
  - PixiJS の Filter を使い、金カットイン時は画面全体を光らせる（Bloom効果）。
  - 文字や画像は Framer Motion ではなく PixiJS の Tweener で制御し 60fps を維持。

## 3. 技術スタック
- Vite + React + TypeScript
- PixiJS (描画エンジン)
- Zustand (状態管理: 通常/リーチ/当たり の管理)
